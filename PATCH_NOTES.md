# 우공실 서비스 변경 이력

이 문서는 우공실에서 새로 추가되었거나 개선된 기능을 날짜별로 정리한 변경 이력입니다. 사용자가 사이트를 이용하면서 달라진 점을 쉽게 확인할 수 있도록 주요 변화만 요약합니다.

## 2026-06-19

### v1.9 화면 테마와 회식맵 사용성을 개선했습니다

#### 화면과 테마

- 다크모드와 라이트모드에서 배경, 글자, 카드, 버튼 색상이 더 자연스럽게 어울리도록 정리했습니다.
- 화면 밝기를 10% 단위로 조절할 수 있는 슬라이더를 추가했습니다.
- 밝기 조절 시 일부 영역만 따로 튀어 보이던 문제를 줄이고, 전체 화면이 함께 어울리도록 보정했습니다.
- 홈 화면의 메인 문구가 더 안정적으로 가운데 배치되도록 정리했습니다.
- 로그인, 회원가입, 마이페이지, FAQ, 게시판, 운세, 문제 풀이 화면의 안내 문구와 버튼명이 더 일관되게 보이도록 다듬었습니다.

#### 필기/실기 문제 풀이

- 필기 문제은행과 실기 문제은행에서 방금 풀었던 문제가 바로 다시 나올 가능성을 줄였습니다.
- 전체 과목 랜덤 풀이에서는 최근에 나온 과목도 가능한 한 피해서 더 다양한 문제를 만날 수 있도록 개선했습니다.
- 문제 수가 부족한 경우에는 자동으로 조건을 완화해, 문제 풀이가 멈추지 않도록 처리했습니다.
- 실기 문제은행과 기출문제 화면의 안내 문구, 답안 입력 안내, 결과 문구, 이미지 확대 버튼 문구를 더 읽기 쉽게 정리했습니다.
- 오답노트의 탭, 필터, 확인창, 버튼 문구를 더 명확하게 정리했습니다.

#### 회식맵

- 장소를 제보하면 관리자 승인 대기 없이 바로 회식맵에 공개되도록 변경했습니다.
- 장소 수정이나 삭제 요청은 기존처럼 검토 후 반영되도록 유지해, 잘못된 정보가 바로 바뀌는 일을 줄였습니다.
- 장소 제보 화면의 깨진 글자와 어색한 문구를 정상 한글 안내문으로 고쳤습니다.
- 지도/후기 버튼명을 실제 사용 흐름에 맞게 `카카오지도/후기 보기`로 정리했습니다.
- 장소 제보 시 카테고리를 직접 입력할 수 있고, 한 글자만 입력해도 관련 카테고리 추천을 볼 수 있도록 개선했습니다.
- 활동 이력 버튼 위치와 활동 이력 팝업의 닫기 버튼을 더 누르기 편하게 정리했습니다.
- 회식맵의 검색창, 지도 영역, 상세 패널, 댓글 입력창이 라이트/다크 모드와 밝기 조절에 더 자연스럽게 맞춰지도록 보정했습니다.

#### 안정성

- 사이트 점검용 health check를 정리해 배포 후 서버와 DB 연결 상태를 더 정확하게 확인할 수 있도록 했습니다.
- 일부 오래된 데이터 구조와 함께 실행될 때 발생할 수 있던 초기 기동 오류 가능성을 줄였습니다.
- 관리자 페이지의 표, 탭, 입력창, 상세 팝업이 다크모드와 라이트모드에서 더 잘 보이도록 보정했습니다.

## 2026-05-30

### v1.7 회식맵 운영 보강

- 회식맵 장소, 댓글, 좋아요, 신고, 수정 제안 데이터를 MySQL 테이블로 관리하도록 구성했습니다.
- 관리자가 회식맵 장소 제보와 수정 제안을 승인, 반려, 삭제할 수 있는 운영 API와 관리자 화면을 보강했습니다.

- 회식맵 페이지 문구와 레이아웃 값을 DB에서 관리할 수 있도록 관리자 설정 기능을 추가했습니다.
- 카카오 지도 JavaScript 키와 Local REST API 키를 분리해 사용하도록 설정하고, 키 점검/REST 테스트/SDK 점검 API를 추가했습니다.
- 회식맵 지도 표시, 장소 검색, 주소 좌표 변환, 카카오 장소 키워드 검색 흐름을 보강했습니다.

## 2026-05-27

### v1.4 운영 공지 및 점검 모드

## 2026-05-24

## 배포 전 확인 기준

배포 또는 저장소 공개 전에는 아래 검증을 기준으로 확인합니다.

```bash
cd backend
npm run check
node server.js
```

```bash
cd frontend
npm run lint
npm run build
```

관리자 페이지의 사용자 관리, 문제/해설 관리, 공지 관리, 화면 설정, 회식맵 승인/반려 기능은 실제 DB와 연결된 상태에서 추가로 확인합니다.
